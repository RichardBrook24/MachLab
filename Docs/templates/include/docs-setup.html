Running Mach Lab on Windows

Installation

  1. Download the Mach Lab Installer for Windows.
  2. Once it is downloaded, run the installer (MachLab-version.exe). This will only take a minute.
  3. By default, Mach Lab is installed under C:\Program Files\Mach Lab for a 64-bit machine.
